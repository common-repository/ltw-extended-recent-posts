Extended version of the recent posts widget.

== Description ==

Extended version of the recent posts widget.

It shows the posts excerpt and has a few extra options like number of words that the excerpt should display,
custom show more characters and to show or hide the date of the post.

== Installation ==

1. Upload `ltw-extended-recent-posts` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Move the widget to the sidebar on the 'Widgets' page in Wordpress
